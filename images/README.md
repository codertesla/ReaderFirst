# Theme Preview Images

The Hugo Themes submission requires two preview images at the repository root `images/` folder, both at a **3:2** aspect ratio:

| File                | Dimensions  | Purpose                                  |
| ------------------- | ----------- | ---------------------------------------- |
| `screenshot.png`    | 1500 × 1000 | Full-screen theme screenshot             |
| `tn.png`            | 900 × 600   | Thumbnail shown in the themes gallery    |

This folder contains the official theme preview images (`screenshot.png`, `tn.png`) used for the Hugo Themes directory.

These images are captured at the required aspect ratio (3:2) and dimensions (1500×1000 and 900×600) to ensure a high-quality presentation in the theme showcase.
